════════════════════════════════════════════════════════════════════════════════
                           🚀 QUICK START GUIDE
════════════════════════════════════════════════════════════════════════════════

STEP 1: Install (First Time Only)
  $ pip install -r requirements.txt

STEP 2: Start Server
  Option A: $ ./run_web.sh
  Option B: $ python web_interface.py

STEP 3: Open Browser
  Go to: http://localhost:5000

STEP 4: Use Interface
  1. Click "Add Workers & Shifts"
  2. Add workers (fill form, click Add Worker)
  3. Add shifts (fill form, click Add Shift)
  4. Click "Proceed to Scheduling"
  5. Click "Generate Schedule"
  6. Review 3 solution options
  7. Click "Download as CSV"

TOTAL TIME: ~10 minutes from start to results

════════════════════════════════════════════════════════════════════════════════
                            📊 API ENDPOINTS (15)
════════════════════════════════════════════════════════════════════════════════

PAGES:
  GET /                    - Home page
  GET /setup              - Setup page (workers & shifts)
  GET /schedule           - Schedule generation page

WORKERS (5 endpoints):
  POST /api/workers       - Add new worker
  GET /api/workers        - List all workers
  GET /api/workers/<id>   - Get specific worker
  PUT /api/workers/<id>   - Update worker
  DELETE /api/workers/<id> - Delete worker

SHIFTS (3 endpoints):
  POST /api/shifts        - Add new shift
  GET /api/shifts         - List all shifts
  DELETE /api/shifts/<id> - Delete shift

SCHEDULING (4 endpoints):
  POST /api/schedule                  - Generate schedules
  GET /api/default-weights            - Get default optimization weights
  GET /api/stats                      - Get statistics
  POST /api/clear-all                 - Clear all data (testing)
